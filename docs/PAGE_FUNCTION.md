# QiaKon KAG 平台 - 页面与功能清单

> **文档类型**：页面与功能清单  
> **版本**：v1.0  
> **日期**：2026-04-28  
> **状态**：初稿  
> **说明**：本文档列出平台所有需要实现的前端页面、功能点及交互要求，与 [FUNC.md](FUNC.md) 的功能清单一一对应。

---

## 一、整体页面架构

```
QiaKon KAG 平台
├── 登录页
└── 主应用框架（左侧导航 + 顶部栏）
    ├── 首页/对话 (/)
    ├── 工作台/仪表盘 (/dashboard)
    ├── 文档管理模块
    ├── 知识图谱模块
    ├── 检索问答模块
    ├── 工作流编排模块（预留）
    ├── 系统管理模块
    ├── 权限管理模块（部门/角色/用户）
    └── 个人中心
```

## 二、认证授权模块

### 2.1 页面清单

| 页面      | 路由         | 功能说明                         |
| --------- | ------------ | -------------------------------- |
| 登录页    | `/login`     | 用户账号密码登录                 |
| 首页/对话 | `/`          | 检索问答对话，直接可用的工作入口 |
| 工作台    | `/dashboard` | 展示系统概览统计                 |

### 2.2 页面功能详述

#### 登录页 (`/login`)

| 功能点       | 交互描述                                                          |
| ------------ | ----------------------------------------------------------------- |
| 账号密码登录 | 输入用户名/密码，调用 POST /api/auth/login，返回 JWT Token 并存储 |
| Token 持久化 | 将 Token 存入 localStorage，支持页面刷新后自动登录                |
| 登录错误提示 | 密码错误/账号不存在时显示具体错误信息                             |
| 登录状态检测 | 已登录用户访问自动跳转至首页/对话页 (`/`)                         |

#### 首页/对话 (`/`)

> **说明**：用户登录后直接进入对话页面，可立即进行检索问答，是平台的核心使用入口。

| 功能点   | 交互描述                                                 |
| -------- | -------------------------------------------------------- |
| 对话界面 | 展示对话历史列表、当期会话，支持多轮对话                 |
| 检索问答 | 输入自然语言问题，系统检索知识库并生成回答               |
| 模型切换 | 顶部可切换当前使用的推理模型                             |
| 历史会话 | 左侧展示历史会话列表，支持创建新会话、切换会话、删除会话 |

#### 工作台/仪表盘 (`/dashboard`)

| 功能点       | 交互描述                                         |
| ------------ | ------------------------------------------------ |
| 统计卡片     | 展示文档总数、图谱实体数、今日问答数、活跃用户数 |
| 最近文档列表 | 展示最近上传的 5 篇文档，支持快速跳转            |
| 最近问答记录 | 展示最近 5 条问答记录，支持点击查看详情          |
| 系统健康状态 | 展示 LLM/数据库/Redis 等组件健康状态指示灯       |

## 二、权限管理模块

### 二、1 页面清单

| 页面       | 路由                 | 功能说明                   |
| ---------- | -------------------- | -------------------------- |
| 部门管理页 | `/admin/departments` | 维护组织架构，支持多级部门 |
| 角色管理页 | `/admin/roles`       | 管理角色及权限矩阵         |
| 用户管理页 | `/admin/users`       | 管理用户账号、角色、部门   |

### 二、2 部门管理页 (`/admin/departments`)

| 功能点       | 交互描述                                   |
| ------------ | ------------------------------------------ |
| 部门树展示   | 左侧树形结构展示部门层级，支持展开/折叠    |
| 新建部门     | 输入部门名称，选择上级部门，提交创建       |
| 编辑部门     | 修改部门名称、调整上级部门                 |
| 删除部门     | 确认提示，删除后该部门用户将失去本部门权限 |
| 部门成员查看 | 点击部门查看该部门下的用户列表             |

**树形结构示例**：
```
├── 公司总部
│   ├── 技术部
│   │   ├── 前端组
│   │   └── 后端组
│   ├── 运营部
│   └── 市场部
```

### 二、3 角色管理页 (`/admin/roles`)

| 功能点           | 交互描述                                                                           |
| ---------------- | ---------------------------------------------------------------------------------- |
| 角色列表         | 展示系统内置角色和自定义角色，含：角色名称、描述、用户数                           |
| 内置角色详情     | Admin、KnowledgeAdmin、DepartmentManager、DepartmentMember、Guest 可查看但不可编辑 |
| 自定义角色       | 支持新建角色，输入角色名称和描述                                                   |
| **权限矩阵配置** | 核心功能：配置角色对各资源的读/写/删权限                                           |

#### 权限矩阵配置（核心功能）

| 资源类别   | 资源项          | 查看     | 新建/编辑 | 删除 | 说明                     |
| ---------- | --------------- | -------- | --------- | ---- | ------------------------ |
| **文档**   | 公开文档        | ✅/❌      | ✅/❌       | ✅/❌  |                          |
|            | 本部门文档      | ✅/❌      | ✅/❌       | ✅/❌  |                          |
|            | 所有文档        | ✅/❌      | ✅/❌       | ✅/❌  | 仅 Admin/KnowledgeAdmin  |
| **图谱**   | 公开实体/关系   | ✅/❌      | ✅/❌       | ✅/❌  |                          |
|            | 本部门实体/关系 | ✅/❌      | ✅/❌       | ✅/❌  |                          |
|            | 所有实体/关系   | ✅/❌      | ✅/❌       | ✅/❌  | 仅 Admin/KnowledgeAdmin  |
| **系统**   | 用户管理        | ✅/❌      | ✅/❌       | ✅/❌  | 仅 Admin                 |
|            | 角色管理        | ✅/❌      | ✅/❌       | ✅/❌  | 仅 Admin                 |
|            | 部门管理        | ✅/❌      | ✅/❌       | ✅/❌  | 仅 Admin                 |
|            | 系统配置        | 查看 ✅/❌ | 编辑 ✅/❌  | -    | 仅 Admin                 |
|            | 审计日志        | 查看 ✅/❌ | -         | -    |                          |
|            | 健康检查        | 查看 ✅/❌ | -         | -    |                          |
| **字段级** | 敏感字段可见性  | ✅/❌      | -         | -    | 如薪资字段仅部分角色可见 |

### 二、4 用户管理页 (`/admin/users`)

| 功能点       | 交互描述                                                     |
| ------------ | ------------------------------------------------------------ |
| 用户列表     | 分页展示用户，展示：用户名、邮箱、部门、角色、状态、最后登录 |
| 筛选/搜索    | 按部门、角色、状态筛选，按用户名/邮箱搜索                    |
| 新建用户     | 打开表单：用户名、邮箱、初始密码、部门（多选）、角色         |
| 编辑用户     | 修改用户信息：部门、角色、状态                               |
| 重置密码     | 管理员可重置用户密码，发送新密码至用户邮箱                   |
| 启用/禁用    | 切换用户状态，禁用后不可登录                                 |
| 批量操作     | 批量启用/禁用/删除                                           |
| 部门归属展示 | 显示用户所属部门树路径                                       |

### 二、5 前端路由权限守卫

| 功能点       | 交互描述                                                                      |
| ------------ | ----------------------------------------------------------------------------- |
| 路由守卫     | 用户访问页面时，前端根据用户角色判断是否有权限访问                            |
| 无权限重定向 | 无权限时跳转至 403 页面或首页，并提示"无权访问"                               |
| 角色受限页面 | Admin 专属页面（如用户管理、角色管理、系统配置）非 Admin 用户不可见侧边栏入口 |
| 登录状态校验 | Token 过期自动跳转登录页，401 响应时清除本地登录状态                          |

### 二、6 前端 UI 元素权限控制

| 功能点       | 交互描述                                                            |
| ------------ | ------------------------------------------------------------------- |
| 按钮级权限   | 根据用户角色显示/隐藏操作按钮（如"删除"按钮）                       |
| 字段级权限   | 表单和详情页中，根据角色显示/隐藏敏感字段（如文档的"访问级别"字段） |
| 列表操作列   | 根据行数据权限动态显示操作按钮（本人/本部门/公开资源）              |
| 下拉选项过滤 | 根据角色过滤下拉选项（如部门选择器，DepartmentMember 只能选本部门） |

### 二、7 API 请求权限注入

| 功能点         | 交互描述                                                             |
| -------------- | -------------------------------------------------------------------- |
| Token 自动携带 | 所有 API 请求自动在 Header 携带 JWT Token                            |
| 权限上下文     | 后端 API 根据 Token 解析用户角色和部门，注入权限过滤条件             |
| 检索权限过滤   | RAG 检索时自动追加 `department_id IN (...) OR is_public = true` 过滤 |

### 二、8 权限状态管理

| 功能点     | 交互描述                                             |
| ---------- | ---------------------------------------------------- |
| 权限 Store | 使用 Zustand 管理当前用户的角色、部门、权限点        |
| 权限初始化 | 登录后调用 `/api/auth/me` 获取完整用户信息和权限列表 |
| 权限广播   | 权限变更时（如角色升级）通知相关组件刷新权限状态     |

## 三、文档管理模块

### 3.1 页面清单

| 页面       | 路由                  | 功能说明                       |
| ---------- | --------------------- | ------------------------------ |
| 文档列表页 | `/documents`          | 分页展示文档，支持筛选和搜索   |
| 文档上传页 | `/documents/upload`   | 上传新文档，填写元数据         |
| 文档详情页 | `/documents/:id`      | 查看文档内容、解析结果、块数量 |
| 文档编辑页 | `/documents/:id/edit` | 编辑文档元数据                 |
| 索引管理页 | `/documents/index`    | 查看索引状态，支持触发重建     |

### 3.2 页面功能详述

#### 文档列表页 (`/documents`)

| 功能点   | 交互描述                                                              |
| -------- | --------------------------------------------------------------------- |
| 分页展示 | 每页 20 条，支持切换页码和每页条数                                    |
| 筛选器   | 按部门（全部/本部门）、文档状态（待索引/索引中/已完成）、时间范围筛选 |
| 搜索     | 按文档标题关键词搜索，实时搜索（300ms 防抖）                          |
| 列表操作 | 每行显示：标题、所属部门、状态标签、上传时间，支持查看/编辑/删除      |
| 批量操作 | 支持批量选择后批量删除                                                |
| 排序     | 支持按上传时间/标题排序                                               |

#### 文档上传页 (`/documents/upload`)

| 功能点       | 交互描述                                                                                                        |
| ------------ | --------------------------------------------------------------------------------------------------------------- |
| 文件选择     | 支持拖拽上传，点击选择；支持 PDF/Word/Markdown/TXT，最大 50MB                                                   |
| 文件格式校验 | 上传前校验扩展名与 MIME 类型，不符合显示错误提示                                                                |
| 上传进度     | 显示上传进度条，支持取消上传                                                                                    |
| 元数据表单   | 包含：文档标题（自动提取或手动输入）、所属部门（选择器）、公开范围（公开/部门内/私有）、访问级别（只读/可引用） |
| 提交确认     | 上传完成后显示成功提示，自动跳转至文档详情页                                                                    |

#### 文档详情页 (`/documents/:id`)

| 功能点       | 交互描述                                                 |
| ------------ | -------------------------------------------------------- |
| 文档信息展示 | 标题、所属部门、创建时间、文档状态、文件大小             |
| 内容预览     | 展示解析后的文本内容，支持分块查看                       |
| 分块列表     | 展示分块结果，每块显示：块序号、内容摘要、向量维度、状态 |
| 索引状态     | 显示当前索引进度（待索引/索引中/已完成/索引失败）        |
| 操作按钮     | 重新解析、重新索引、下载原文件、编辑、删除               |

#### 文档编辑页 (`/documents/:id/edit`)

| 功能点     | 交互描述                                   |
| ---------- | ------------------------------------------ |
| 元数据编辑 | 可编辑：标题、所属部门、公开范围、访问级别 |
| 保存/取消  | 保存后触发增量索引                         |

#### 索引管理页 (`/documents/index`)

| 功能点   | 交互描述                             |
| -------- | ------------------------------------ |
| 索引队列 | 展示待索引/索引中的文档列表及进度    |
| 索引操作 | 支持批量重试失败任务、全量重建索引   |
| 索引统计 | 展示总块数、索引成功率、平均索引耗时 |

## 五、检索问答模块

### 5.1 页面清单

| 页面       | 路由                     | 功能说明         |
| ---------- | ------------------------ | ---------------- |
| 问答对话页 | `/retrieval/chat`        | RAG 问答对话界面 |
| 对话历史页 | `/retrieval/history`     | 历史对话记录列表 |
| 对话详情页 | `/retrieval/history/:id` | 查看单条对话详情 |

### 5.2 页面功能详述

#### 问答对话页 (`/retrieval/chat`)

| 功能点           | 交互描述                                                  |
| ---------------- | --------------------------------------------------------- |
| 对话输入         | 底部输入框，支持回车发送，支持 Shift+回车换行             |
| **推理模型切换** | **顶部模型下拉框，支持随时切换当前对话使用的推理模型**    |
| 流式回复         | LLM 回复以流式方式逐字显示，显示 Loading 状态             |
| 来源标注         | 回复完成后，文档来源以脚注/气泡形式标注，支持点击查看原文 |
| 上下文关联       | 同一对话内自动携带历史消息（受 Token 限制自动裁剪）       |
| 检索结果展示     | 可折叠展示本次问答引用的文档块列表                        |
| 复制/反馈        | 支持复制回复内容，对答案满意度评分（赞/踩）               |
| 清空对话         | 点击清空当前对话，重新开始                                |

> **动态模型切换**：用户可在对话过程中随时切换推理模型，切换后当前对话上下文保留，但后续回复将使用新的模型生成。

#### 对话历史页 (`/retrieval/history`)

| 功能点     | 交互描述                               |
| ---------- | -------------------------------------- |
| 对话列表   | 展示所有历史对话，按最后更新时间倒序   |
| 列表项信息 | 对话标题（首句预览）、创建时间、消息数 |
| 搜索       | 按对话内容关键词搜索                   |
| 操作       | 点击进入对话详情，滑动/按钮删除对话    |

#### 对话详情页 (`/retrieval/history/:id`)

| 功能点   | 交互描述                                 |
| -------- | ---------------------------------------- |
